# Fun Rent - Strona wizytówka

Nowoczesna, responsywna strona wizytówka dla firmy event planningowej zajmującej się wynajmem sprzętu, animacjami i kompleksową obsługą eventów.

## Technologie

- React 18
- TypeScript
- Vite
- React Router

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev
```

## Budowanie

```bash
npm run build
```

## Logo

Projekt wymaga pliku logo w formacie PNG w katalogu `public/logo.png`. Logo powinno być w formacie PNG zgodnie z wymaganiami projektu.

## Struktura projektu

- `src/pages/` - Strony aplikacji (Home, Katalog, O nas, Kontakt, Galeria)
- `src/components/` - Komponenty UI (Navbar, HeroCarousel, CatalogCard, TestimonialsSlider)
- `src/mock/` - Dane mockowe (funrent.data.ts)
